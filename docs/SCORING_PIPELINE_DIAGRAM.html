Tali — end-to-end candidate scoring pipeline

Purple spine = the scoring engine (changes the score / decision). Slate = the integrity / fraud layer (advisory flags only).

This is the CV / match pipeline. The separate assessment report shows one canonical scorecard — the 5 dimensions (the 4 Ds + Deliverable): Delegation, Description, Discernment, Diligence, Deliverable. See SCORING_SCORECARD.md. (The "dimensions" in Stage 2 below are per-requirement CV grades.)

End-to-end candidate scoring pipeline for Tali: artefacts (CV, job spec, requirements, Workable, knowledge graph, GitHub) flow through ingest, a pre-screen gate, the holistic Sonnet score, a synchronous integrity layer, the deterministic decision, and an async corroboration step. The purple spine changes the score and decision; the slate integrity layer is advisory flags only.
